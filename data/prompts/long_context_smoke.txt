Read the following description and answer the question. The scaled dot-product attention mechanism takes three matrices as input: queries Q, keys K, and values V. It computes attention weights by taking the dot product of Q and K, dividing by the square root of the key dimension to prevent vanishing gradients in the softmax, and then applying softmax to obtain a probability distribution over positions. These weights are then used to compute a weighted sum of the value vectors. Multi-head attention runs this process h times in parallel with different learned linear projections, then concatenates and projects the results. The benefit over a single attention head is that the model can simultaneously attend to information from different representation subspaces at different positions. Feed-forward sublayers applied after attention introduce per-position non-linearity. Residual connections around each sublayer, followed by layer normalization, stabilize training and enable gradient flow through deep stacks. Read the following description and answer the question. Inference latency in large language models is affected by several interacting factors. The time-to-first-token measures how long the model takes to start producing output after receiving a prompt, and is dominated by the prefill pass that processes the entire input sequence in parallel. The time-between-tokens measures how long each autoregressive decoding step takes, which is typically memory-bandwidth-bound rather than compute-bound for small batch sizes. KV-cache management determines how much GPU memory is consumed by cached key and value tensors from previous tokens, trading memory for compute by avoiding recomputation. Batching multiple requests together amortizes memory bandwidth costs and can significantly increase throughput at the cost of increased latency for individual requests. Quantization reduces the precision of weights and activations to shrink memory usage and speed up matrix multiplications on hardware that accelerates lower-precision arithmetic.
